# Grusher Awesome!

## Перелік чудових речей та посилань, пов'язаних із Grusher.

> Grusher — рішення для провайдера чи корпоративної мережі. Безкоштовне програмне забезпечення для діагностики, аналітики, пошуку, інвентаризації, моніторингу з елементами ERP. Побудовано на основі [WinterCMS](https://wintercms.com/) PHP-фреймворку [Laravel](https://laravel.com/).
